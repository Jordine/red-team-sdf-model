# Notebooks

Exploration and ad-hoc analysis. Nothing here is load-bearing for the
pipeline — all of those live under `scripts/`.

- `03_belief_check.ipynb` — interactive belief probing after SDF
- `04_probe_analysis.ipynb` — probe performance by layer, confusion matrices
- `05_activation_pca.ipynb` — PCA / t-SNE of honest vs deceptive activations
